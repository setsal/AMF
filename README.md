# go-AMF

Automatic message forwarding

## Install

```
git clone https://github.com/setsal/go-AMF.git
cd go-AMF
make install
```

After install, the `go-AMF` binary will build in your $GOPATH/bin

## Build
```
make build
```

The `go-AMF` binary will build in the project current dir
